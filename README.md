## Getting Started

These instructions will help you set up and run the project on your local machine.

### Prerequisites
React + Vite
Node

### Installation

1. [Step 1]
    ```bash
    npm install
    ```

2. [Step 2]
    ```bash
    npm start (or) npm run start
    ```

Follow the Link: http://localhost:5173/

## Usage

### Uploading the Cats
